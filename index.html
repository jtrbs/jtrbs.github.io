CS240 Project1

By Tairan Jin

In this project, I use XQuery to write queries on v-emps.xml and v-depts.xml to solve following problems. I use baseX to develop and test. Also I have solved two problems with extra credits successfully.

Note: All the results are visualized with stylesheets for Ectra Credit!

Problem1

Problem: Selection and Temporal Projection. Print the salary history of employee "Anneke Preusig."
Solution: You can see the query first.xq and the result Q1_result.

Problem2

Problem: Temporal Snapshot. Print the name, salary and department of each employee who, on 1995-01-01, was making more than $80,000.
Solution: You can see the query second.xq and the result Q2_result.

Problem3

Problem:Temporal Slicing. For all departments, print their histories for the period starting on 1994-05-01 and ending 1996-05-06.
Solution: You can see the query third.xq and the result Q3_result.

Problem4

Problem:Duration after Coalescing. For each employee, find the longest period (or periods) during which he/she went with no change in salary: for each employee print the period(s), his/her name and the actual salary.
Solution: You can see the query forth.xq and the result Q4_result.

Problem5

Problem:Temporal Join. For each employee find the longest consecuitive period in which he/she worked in the same department and the same department manager: Print the employee number, his/her department number and name, his/her manager number, and the period.
Solution: You can see the query fifth.xq and the result Q5_result.

Problem6

Problem:Temporal Avg. Print the history of the average salary for (i) the whole company and (ii) each job title.
For 6-1 Solution: You can see the query sixth-1.xq and the result Q6_1_result.
For 6-2 Solution: You can see the query sixth-2.xq and the result Q6_2_result.

Problem7

Problem:Temporal Max. For the employees in department d005, find the maximum of their salaries over time, and print the history of such a maximum.
Solution: You can see the query seventh.xq and the result Q7_result.
